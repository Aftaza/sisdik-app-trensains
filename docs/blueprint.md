# **App Name**: SISDIK App

## Core Features:

- User Authentication: Login page with email and password inputs.
- Dashboard Overview: Main dashboard with a search bar, user profile, and key performance indicators for student discipline.
- Violation Form: Modal form for creating new student violation records with searchable student selection and violation details.
- Student Profile: Detailed student profile page with violation history and total violation points.
- Student filter: Filter list of students based on search queries to quickly identify violators
- Violation AI Categorization : Utilize AI tool to categorize the violations type after a user inputs information. This tool will extract and reason based on provided information about violations type, assigning categories that fit within your desired set. 

## Style Guidelines:

- Primary color: Calming blue (#4A90E2) for buttons and accents, reflecting trust and stability.
- Background: Neutral white (#FFFFFF) for primary backgrounds, creating a clean and open feel.
- Secondary background: Light grey (#F4F7FA) for content sections, providing visual separation.
- Text color: Dark grey (#333333) for text, ensuring high readability.
- Success notification: Touch of green (#50E3C2) for success alerts.
- Font: 'Inter' or 'Poppins' (sans-serif) for a modern, clean look. Headings in bold, body text in regular weight for optimal readability.
- Card-based layout with rounded corners (8px border-radius) and consistent padding for a spacious and user-friendly design.
- Top navigation bar including app logo ('SISDIK'), a search bar, and a user profile icon.
